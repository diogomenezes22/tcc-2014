Como Funciona?

O sistema será responsável por controlar a dinâmica do jogo, dos jogadores e seu andamento a cada rodada. Caberá ao administrador controlar e criar o jogo, assim como adicionar os jogadores que participarão. Cabe ao sistema sortear, antes de cada jogo, quais serão as casas do tabuleiro que corresponderão às prendas.

Após a criação da partida, as rodadas se alternarão entre os jogadores cadastrados pelo administrador, e, para cada rodada, o jogador terá o direito de jogar o dado. Ao parar numa casa do tabuleiro correspondente a casa de pergunta, o sistema irá sortear uma pergunta pré-cadastrada no banco de dados para o jogador responder.

Se responder corretamente, avançará o número de casas correspondente ao número sorteado no dado, se errar a pergunta, o sistema não permitirá que avance o número de casas sorteado no dado.

Ao parar numa casa correspondente a prenda, o sistema irá sortear uma prenda e deverá pagá-la diante da turma e o mesmo não avança mais nenhuma casa na rodada.

Quem vence?

Vence o primeiro jogador que alcançar a última casa do tabuleiro.
